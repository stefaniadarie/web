Index

Pagina "Index" este pagina principala a sit-ului. Impreuna cu formularul de inscrieri, autentificare, pagina "Despre noi" si "Ajutor" constituie singurele pagini la care au acces persoanele ce nu si au creat inca un cont pe platforma.

Sectiunea <body> contine butoane cu nume sugestive atribuite pentru fiecare dintre paginile mentionate anterior. In centrul paginii se afla doua element asemanatoare cu o fereastra care contin numele paginii si o scurta descriere a acesteia(momentan se afla in lucru). Ambele sunt construite cu <div class="window">, diferentele apar doar la continut si pozitionare, cea din urma fiind realizata cu ajutorul unor id-uri de overlay.

Despre noi

Pagina contine o descriere a sit-ului(in lucru), date de contact si numele realizatorilor. Aceste informatii sunt incadrate intr-o fereastra, la fel ca si in cazul Indexului. Suplimentar, sunt adaugate noi butoane: Meniu, Cont si bara de cautare. La accesarea meniului se va deschide un panou lateral de unde se vor putea accesa mai multe sectiuni(in lucru). La implementarea lui am utilizat si JavaScript. Butonul "Cont" va duce la pagina ce contine datele personale ale utilizatorului(in lucru). De asemenea, mai este prezent si butonul "Ajutor".

Ajutor

Aceasta pagina este impartita in doua, in prima jumatate se afla o fereastra mai mica cu o poza incadrata in ea, iar in ce a de a doua se afla mai multe sectiuni ce vor cuprinde diferite probleme intalnite de utilizatori, indicatii, etc. Momentan in sectiunile create am inclus acest Scholarly si mockup-ul. Totodata, in partea superioara a paginii se afla butoanele pentru Meniu, Cont si bara de cautare.

Inregistrare

Scopul paginii de inregistrare este de a permite utilizatorilor sa-si creeze un cont si sa acceseze continutul oferit de platforma In<form>atiX. Pagina este structurata in doua ecrane, fiecare cu propriul sau set de informatii si formulare.

Incepand cu sectiunea <!DOCTYPE html>, se specifica ca acesta este un document HTML5. In urmatoarele linii se specifica un set de metadate, inclusiv codificarea caracterelor, titlul paginii si un fisier CSS extern pentru stilizarea paginii. In sectiunea <body> se gaseste continutul principal al paginii. Acesta include un buton de navigare inapoi la pagina principala, o componenta care include cele doua ecrane ale paginii de inregistrare, un buton pentru navigarea catre pagina "Despre noi" si un buton pentru a obtine ajutor.

Componenta care contine cele doua ecrane de inregistrare este construita folosind div-uri cu clase specifice pentru stilizare. Primul ecran, <div class="screen1 margin-left">, este compus dintr-un antet si un corp. Antetul contine trei puncte care tin strict de aspect, iar corpul contine un text de intampinare pentru utilizator. Al doilea ecran, <div class="screen2 margin-right">, contine, de asemenea, un antet si un corp. Antetul contine acelasi set de trei puncte care tin strict de aspect. Acest formular contine doua sectiuni, una pentru datele personale si una pentru datele de autentificare. Fiecare sectiune contine un set de etichete si campuri de introducere.

Autentificare

Aceasta pagina contine doua ecrane, cu o caseta pentru header si una pentru body. Prima caseta contine un mesaj de salut si o gluma, iar a doua caseta contine formularul pentru autentificare, unde utilizatorii trebuie sa introduca adresa lor de email si parola.

Pagina de autentificare a acestei platforme este creata in HTML, si este structurata in doua ecrane. Prima caseta contine un mesaj de salut si o gluma, iar a doua caseta contine formularul pentru autentificare.

Codul HTML pentru pagina de autentificare a platformei In<form>atiX este structurat in urmatorul mod:

  1. Sectiunea head: In aceasta sectiune se afla informatii despre pagina, cum ar fi meta tag-urile pentru encoding si titlul paginii. De asemenea, se face referire la un fisier CSS pentru a formata pagina.
  2. Sectiunea body: Aceasta sectiune contine continutul principal al paginii. In interiorul corpului paginii, se afla butonul de intoarcere la pagina principala, o diviziune care contine cele doua ecrane, butoane pentru "Despre noi" si "Ajutor", si o diviziune pentru un cerc decorativ.

Primul ecran contine un mesaj de salut si o gluma. In interiorul acestei casete se afla o diviziune pentru header si una pentru body. Header-ul contine trei puncte care tin doar de aspect. In body, se afla un mesaj de salut ("Bine ai revenit pe In<form>atiX!") si o gluma despre programatori.

Cel de-al doilea ecran contine formularul de autentificare. Acesta este structurat intr-un header si un body. In body, se afla un titlu pentru formularul de autentificare ("Autentificare") si doua campuri de intrare: unul pentru adresa de email si unul pentru parola. Formularul are si un buton de submit pentru a trimite datele introduse.
