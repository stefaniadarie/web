Motivatie

Proiectul InformatiX vine sa raspunda nevoii tot mai mari de dezvoltare a abilitatilor de programare a elevilor de liceu, intr-un mod distractiv si interactiv. Prin intermediul acestei aplicatii web, elevii vor putea lucra cu probleme concrete si utile, invatand astfel intr-un mod mai eficient si placut. De asemenea, InformatiX poate fi o unealta utila si pentru profesori, care vor putea propune probleme si teme personalizate, adaptate la nivelul si nevoile elevilor lor, precum si pentru evaluarea acestora. Astfel, profesorii vor putea urmari progresul individual si colectiv al elevilor lor, identificand punctele tari si punctele slabe si oferind feedback specific.

Un alt avantaj al aplicatiei InformatiX este reprezentat de posibilitatea exportului si importului problemelor intr-un format open, ceea ce faciliteaza colaborarea intre diferite aplicatii web de acest gen, precum si posibilitatea de a crea o comunitate de programatori si elevi mai extinsa si mai activa.

Sistemul de evaluare, rapoartele si statisticiile oferite de InformatiX reprezinta o modalitate eficienta de a urmari performanta elevilor si de a identifica problemele si nevoile specifice, astfel incat acestia sa poata fi indrumati spre imbunatatirea abilitatilor lor de programare.

Deci InformatiX vine sa ofere o solutie utila si eficienta pentru dezvoltarea abilitatilor de programare a elevilor de liceu si imbunatatirea procesului de predare-invatare in acest domeniu, cu beneficii pentru elevi, profesori si intreaga comunitate de programatori.

Pagina principala

Este prima pagina a acestei aplicatii si avem urmatorele: pe centru avem o caseta cu o scurta descriere a aplicatiei noastra, deasupra are cele doua butoane (Login si Register), iar dedesubt avem butoanele de About us si Help.

About us (despre noi)

Pagina contine o descriere a aplicatiei noastre, date de contact si numele realizatorilor. Aceste informatii sunt incadrate intr-o fereastra.

Help (ajutor)

Aceasta pagina este destinata spre a ajuta un utilizator sa foloseasca corespunzator aceasta aplicatie.

Register (inregistrare)

Scopul paginii de inregistrare este de a permite utilizatorilor sa-si creeze un cont si sa acceseze continutul oferit de platforma In<form>atiX.

Login

Aceasta pagina contine un form unde utilizatorul trebuie sa introduca numele de utilizator si parola pentru a se loga pe aceasta platforma.

Codul HTML pentru pagina de autentificare a platformei In<form>atiX este structurat in urmatorul mod:

>
  1. Sectiunea header
  2. Sectiunea meniu Aceasta sectiune contine meniul corespunzator fiecarui tip de user (student, teacher, admin).
    3. >
  3. Compnenta respectiva (problem, homework etc)
  4. Sectiunea de footer In aceastea sectiune avem doua cercuri decorative, butonul de Help si butonul de About us.

Meniul pentru profesor

Meniul unui profesor este format din:

  • My classes -> aici poate vedea toate clasele sale si poate adauga si clase noi
  • Problems -> aici poate vedea toate problemele existente pe aceasta platforma si poate adauga si alte probleme noi
  • My problems -> in aceasta sectiune profesorul poate vedea doar problemele propuse de el si poate adauga probleme noi
  • Reports
  • Logout

Meniul pentru elev

  • My classes -> aici elevul poate vedea clasele din care face parte
  • Problems -> in aceasta sectiunea elevul poate vedea toate problemele existente si poate adauga propria solutie la o problema random
  • Reports -> aici elevul ar fi trebuit sa vada un clasament
  • Logout

Meniul pentru elev

  • Pendind problems -> adminul vede problemele care au fost postate de profesori si urmeaza ca acesta sa dea Accept sau Decline problemei respective
  • Import problems
  • Export problems
  • Reports
  • Logout

Paleta de culori

  • yellow: #ffce38;
  • blue: #004A98;
  • white: #f4f4f4;
  • gray1: #cdcdcd;
  • gray2: #cccccc;
  • gray3: #818181;
  • gray4: #4d4d4d;
  • black: #000000;

Descompunând paleta de culori mai detaliat, se poate observa că albastrul este folosit ca o culoare de accent și este utilizat în butoane, link-uri și alte elemente interactive din design-ul aplicației. Albastrul închis (codul #004A98) este o culoare puternică, care sugerează stabilitate și profesionalism. Este o alegere potrivită pentru o aplicație educațională, deoarece sugerează încredere și seriozitate.

Culoarea galbenă (#ffce38) este folosită pentru a adăuga un contrast plăcut și cald în design-ul aplicației. Galbenul este o culoare luminată, care sugerează optimism și energie, iar în contextul educației poate sugera entuziasm și creativitate.

Albul (#f4f4f4) este folosit ca o culoare de fundal, ceea ce sugerează claritate și simplitate. Albul face ca textul și alte elemente ale interfeței să iasă în evidență, permițând utilizatorilor să se concentreze mai ușor asupra conținutului.

Nuanțele de gri (#cdcdcd, #cccccc, #818181 și #4d4d4d) sunt utilizate pentru a oferi contrast și adâncime design-ului. Griul este o culoare neutră, care poate sugera stabilitate și echilibru. De asemenea, griul poate fi asociat cu tehnologia și modernitatea, ceea ce este potrivit pentru o aplicație web ce are ca scop învățarea programării.

Negrul (#000000) este utilizat pentru a adăuga un accent puternic și elegant în design. Este o culoare puternică, care sugerează autoritate și eleganță. Negrul poate fi utilizat pentru a sublinia elementele importante din design sau pentru a crea un contrast puternic cu alte culori din paleta.

În concluzie, această paletă de culori a fost aleasă cu grijă pentru a sugera o atmosferă academică și modernă, care poate fi atractivă pentru utilizatorii din mediul educațional. Culorile alese au fost folosite cu atenție pentru a crea contrast și adâncime design-ului, în timp ce păstrează un aspect simplu și clar, ceea ce este esențial pentru o aplicație web cu un scop educațional.
